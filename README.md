This program solves the problem of students having no understanding of which grades they get and how it relates to their GPA. Some students may not understand the difference between Honors, AP, and general class grade weighting. In this program, you are able to see these differences between each of the grades from an A to an F. For example, for a grade of C, an AP is a 3.0, an honors is a 2.5, and general is 2.0. Once you find all of the GPA’s from your grades, you can add them up and divide it by the number of classes to find your GPA.

Instructions: To use the program, you have to click on whatever grade you want from A to F. Then, on the next screen you can see the weighting of each type of class which is AP, Honors, and General. Once you find which weighting it is, you can go through how ever many classes you have to find all the GPA correlations. This will allow you in a calculator to add up all of the GPA's you found by the number of classes you have.
